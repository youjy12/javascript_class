01variable(변수)복습과 유의사항

  • var : 자바스크립트 es6문법 이전에서 사용하던 변수 키워드
  • let : es6문법에서 새롭게 나온 변수 키워드로서 변하는 값을 저장할 때 사용.
  • const : es6문법에서 새롭게 나온 변수 키워드로서 고정된 값을 저장할 때 사용.

var키워드는 변수선언을 여러번 해도 오류가 나지 않는다.

유의사항

  1. 전역변수는 최소한으로 사용한다.
  2. 변수는 함수의 시작부분에서 선언한다.
  3. for문에서는 카운터 변수를 선언할 때 var예약어를 사용하지 않는다.
    var i;
    for(i=1;i<=10;i++){
      출력할내용
    }
    
    for(let i=1;i<=10;i++){
      출력할 내용
    }
